خطوات ربط التطبيق بـ Firebase

1

إنشاء مشروع Firebase

اذهب إلى Firebase Console وأنشئ مشروعاً جديداً باسم "giedo-digital-mall"

2

إضافة تطبيق Android

في إعدادات المشروع، اضغط على "Add app" واختر Android

Package name: com.giedo.digitalmall
3

تنزيل google-services.json

حمّل الملف وضعه في مجلد android/app/

4

تشغيل FlutterFire CLI

# تثبيت CLI
npm install -g firebase-tools
dart pub global activate flutterfire_cli

# إعداد المشروع
flutterfire configure
5

تفعيل الخدمات

  • Authentication → Email/Password
  • Firestore Database → Test mode
  • Storage (اختياري)

users/ بيانات المستخدمين
orders/ الطلبات والمشتريات
